# 茶思屋 Markdown 发布助手（Chrome浏览器扩展）

## 📖 简介

**茶思屋 Markdown 发布助手** 是一款简化 [黄大年茶思屋](https://www.chaspark.com) 网站发布操作的 Chrome 浏览器扩展。它能够将本地 Markdown 文件及其引用的图片自动上传至茶思屋编辑器，并一键完成内容的自动填充，显著提升发布效率。

### ✨ 主要功能

- � **智能快捷进入**：点击插件图标，自动为您打开发布页面
- �📝 **全自动解析**：自动读取并解析 Markdown 内容
- 🖼️ **图片一键上传**：自动上传本地图片至茶思屋 CDN 并替换链接
- 🎨 **完美适配**：将 HTML 内容自动注入编辑器并保持排版
- � **双模式支持**：支持「单个文件」选择与「项目目录」一键导入

---

## 🔧 安装步骤

1. **获取代码**：下载本项目的代码到本地。
2. **加载扩展**：
   - 打开 Chrome，访问 `chrome://extensions/`
   - 开启右上角的「**开发者模式**」
   - 点击「**加载已解压的扩展程序**」，选择本项目所在的文件夹
3. **固定插件**：点击拼图图标 🧩，将「茶思屋 Markdown 发布助手」固定📌到工具栏。

---

## 📚 使用指南

### 快捷发布流

通常情况下，您只需执行以下步骤：

1. **点击插件图标**：无论原本在哪个页面，插件都会带您直接前往 [内容发布页面](https://www.chaspark.com/#/hotspots/publish)。
2. **选择导入方式**：
   - **方式 1：选择文件**：适合 Markdown 文件与图片分别存放的情况。
   - **方式 2：选择项目目录**：适合整个项目都在同一个文件夹的情况，插件会自动识别首个 `.md` 文件及相关资源。
3. **开始执行**：点击「开始上传并填充」，片刻后内容将自动呈现在编辑器中。

### 前置要求

- **账号登录**：请确保已在浏览器中登录 [Chaspark](https://www.chaspark.com)。
- **路径引用**：Markdown 中的图片请使用**相对路径**（例如 `![](assets/pic.png)`），以便插件正确识别。

---

## ⚠️ 注意事项

- **编辑器状态**：如果点击插件后页面重定向，请在页面完全加载后再点击一次插件图标以唤起操作面板。
- **动态填充**：插件会尝试将转换后的内容直接填入 TinyMCE 编辑器中。
